# Ambient Agency

The next interface shift probably looks less like a new app category and more like disappearance. The useful parts of software move closer to context: memory, intent, delegation, verification.

Screens will remain, but fewer workflows should require treating the screen as the destination. The system should understand the work well enough to carry state across tools, surfaces, and time.

- less navigation
- more continuity
- fewer commands that only exist because the software forgot what you meant
